# Open-source references

This folder is for reference-only open-source review.

Rules:

- Do not import GPL/AGPL code into production source without a written legal/licensing decision.
- Do not copy UI, branding, assets, sounds, product identity, demo data or metadata from reference projects.
- MIT dependencies may be used if documented in `docs/legal/source-usage.md`.
- Any copied/adapted code must be listed in `docs/legal/source-usage.md`.

Fetch references with:

```bash
pnpm fetch:references
```
