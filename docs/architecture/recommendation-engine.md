# Recommendation Engine

V1 ranking is rule-based:

- watch time
- completion rate
- rewatch
- like rate
- comment rate
- follow conversion
- skip rate
- report rate
- block rate
- freshness
- creator quality score
- category/hashtag match
- language/region
- safety score
- boost score

Rules:

- Reported/unsafe content is downranked.
- Blocked creators are hidden.
- Paid boosts never bypass moderation.
- New creators get a controlled cold-start boost.
