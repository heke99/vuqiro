# Batch 001 Plan — Vuqiro Foundation

## Goal

Create the first Vuqiro product foundation for mobile and superadmin.

## Included

- Expo React Native app foundation
- Next.js superadmin foundation
- Shared types
- Shared mock data
- Shared design tokens
- Monetization scaffolding
- Creator subscription and coins UI
- Moderation UI foundation
- Payout/admin UI foundation
- Legal/product/architecture documentation
- Initial Supabase schema draft

## Not included yet

- Real auth
- Real video upload
- Real RevenueCat purchases
- Real Stripe Connect payouts
- Real recommendation engine
- Real moderation automation
- Store submission

## Open source

MIT dependencies/reference may be used. GPL/AGPL projects remain reference-only unless Diversa Solutions LLC makes a written license decision.
