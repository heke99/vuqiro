# Vuqiro Roadmap

## Batch 1

Mobile/admin foundation with mock data, original design, monetization scaffolding and legal/architecture docs.

## Batch 2

Supabase auth, profiles, creators, videos table, basic backend API.

## Batch 3

Video upload, processing provider decision, HLS playback and thumbnail generation.

## Batch 4

RevenueCat integration, Apple/Google product mapping, webhooks, entitlement verification and restore purchases.

## Batch 5

Creator memberships, locked content, wallet transactions, coins spending and creator revenue ledger.

## Batch 6

Stripe Connect onboarding, payout accounts, payout holds and payout release flow.

## Batch 7

Recommendation engine, watch events, ranking, boosts and safety scoring.
